# JonStage - Yoroduwemaki(万絵巻) Official Website

## Overview

This repository contains the source code for Yoroduwemaki(万絵巻) official website. It is designed as a high-performance
static website to introduce the activities and performances of the theater company.

## Key Features

- Performance Schedule
- Cast & Crew Profiles
- Blog/News Updates

## Used by

- [Astro](https://astro.build) - Web framework
- [AsagaoUI](https://asagaoui.com) - Styling

## Structure

```text
jonstage/
├── src/
├── public/
└── astro.config.mjs
```
