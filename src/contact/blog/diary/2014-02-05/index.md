一回生のれのんと申します。
初めてのブログ担当ってことで緊張してます。できてるかな？

今日は試験期間明けの最初の稽古で、ブレストとして時間をかけてこの公演を残り少ない時間でどのようにして行きたいかということを一人一人話し合いました。お互いの真剣な意見のぶつかり合いによって、さらに公演に向けての団結力が高まったなー！という意気込みをひしひしと感じました！
(※あまりの真剣な気迫に思わず稽古場の写真を撮ることを忘れるほどでした！申し訳ないです！m(\_ \_)m )

残り限られた時間ですが
「自分達のために、お客さんのために」
頑張って満足のいく劇を完成させていきましょう！

れのんでした！
